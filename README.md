# 🎓 Student Manager — 学生信息管理系统

## 📖 项目简介

基于 **Java Web + MyBatis** 的学生信息管理系统，实现学生信息的填报、浏览、查询统计及增删改查操作。

用户可在浏览器端填写学生基本信息并提交至后台数据库，支持按学院、专业进行查询统计，同时提供完整的数据增删改查功能。

---

## ✨ 功能特性

- ✅ 学生信息在线填报（学号、姓名、性别、出生日期、政治面貌、身份证号、学员代码、专业代码、年级、家庭住址、联系手机、邮编）
- ✅ 表单校验：必填项未填时禁止提交
- ✅ 浏览全部已提交的学生信息
- ✅ 按学院、按专业进行查询统计
- ✅ 学生数据的增删改查（CRUD）操作

---
